## Usage

## Requirements

An `nvmrc` file is included if using [nvm](https://github.com/creationix/nvm).

- Node 6.13.0
- Mongo 4.4.1

## Development

First time set up:

Run seeding scripts (mongo must be installed and running)
```
npm run seed
```

Start server
```
npm run server:dev
```

### Installing Dependencies

From within the root directory:

```sh
npm install -g webpack
npm install
```
